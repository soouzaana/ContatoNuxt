## Descrição
É uma aplicação que simula uma lista de contatos que permite editar e exluir. Esse projeto possui fins didáticos e foi desenvolvido durante o estudo de Nuxt.js 3.

## Tecnologias Utilizadas
- HTML
- Tailwind
- Vue.js 3
- Nuxt.js 3
- JavaScript
